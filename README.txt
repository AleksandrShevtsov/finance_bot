Финальный виртуальный комплект с возможностью позже подключить реальный депозит.

Что есть:
- Binance поток сделок
- стакан через bookTicker imbalance
- open interest Bybit
- паттерны
- уровни поддержки/сопротивления
- SL/TP по уровням
- partial + BE при +30% к марже
- обычный partial
- trailing
- reverse signal exit
- фильтр тренда
- фильтр волатильности
- шаблон executor для реального режима

Зависимости:
pip install websocket-client requests

Запуск:
python main.py
